## 🚀 快速开始

### 方法一：直接打开文件
1. 在文件浏览器中找到项目文件夹
2. 双击 `index.html` 文件即可在浏览器中打开

### 方法二：使用本地服务器（推荐）
```bash
# 使用 Python 3
python -m http.server 8000

# 或使用 Python 2
python -m SimpleHTTPServer 8000

# 使用 Node.js (需要安装 http-server)
npx http-server

# 使用 PHP
php -S localhost:8000
```

然后在浏览器中访问 `http://localhost:8000`
